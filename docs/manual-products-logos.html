Author

Jason Wallin, OEDA-GIS Manager

OEDA logos are available as .svg files, which Quarto handles natively for HTML files. These are vector graphics files with transparent backgrounds, so they should be able to scale well and be legible in most settings.

You can right click on any of the images below and select save as to save them locally on your machine.
